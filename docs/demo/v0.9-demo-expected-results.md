## 数据源与口径

- 工作簿：`docs/test_datasets/simple/01_门店月度销售.xlsx`
- 工作表：`月度销售`

## 演示产物验收

首问应生成：

1. 一份最终报告，明确给出完整排名及单位。
2. 一张按销售额从高到低排列的门店排名图。
3. 一份 CSV，至少包含 `排名`、`门店`、`上半年销售额(万元)`。
4. 一份 XLSX，至少包含与 CSV 相同的五行排名数据。

追问应生成：

1. 明确给出 Top 2 门店、销售额和它们相对第三名的差值。
2. 一份只包含两家门店的 CSV。

## 判定规则

- 门店全名必须与原始工作簿一致，不能用虚构的地区或商圈名称替代。
- 允许报告使用“北京店”等简称，但结构化产物必须保留完整门店名。
- 展示可四舍五入到 1 位小数；下载文件中的原始数值不得因 UI 格式化而改变。
- 图表颜色、文件名和报告措辞可以变化；排名、数值、单位和行数不能变化。

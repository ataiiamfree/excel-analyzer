# ChatExcel 📊

你的 Excel 智能分析助手。上传文件，用自然语言提问，自动生成分析报告和图表。

### 快速开始

1. **上传文件** — 点击下方上传 `.xlsx` 或 `.xlsm` 文件
2. **提出需求** — 例如："各部门销售额排名，画柱状图"
3. **查看结果** — 报告、图表、明细表一键下载
4. **继续追问** — 无需重新上传，直接追问即可
5. **换文件** — 随时发送新 Excel 文件切换数据

### 能力范围

- 数据汇总、分组统计、排名
- 趋势分析、同比环比
- 相关性分析、异常检测
- 图表生成（折线、柱状、饼图、散点图等）
- 跨 Sheet 关联分析
- 导出分析结果为 Excel/CSV
